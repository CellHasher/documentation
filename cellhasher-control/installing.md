# Installing

{% hint style="info" %}
Cellhasher Control is currently in Beta.
{% endhint %}

## 1. Download Latest Release&#x20;

1. Visit: [https://github.com/CellHasher/Beta-Cellhasher/releases](https://github.com/CellHasher/Beta-Cellhasher/releases)
2. Under Assets, download _Cellhasher.Controller.exe_
3. Proceed by following the installaiton process



## 2. Download Additional Requirements <a href="#scrcpy" id="scrcpy"></a>

1. Open Cellhasher Control and head to  [network.md](tabs-walkthrough/network.md "mention") tab.
2. &#x20;Use the quick download buttons the additional requirements necessary to use certain features.



## 3. Set-Up ADB Keys (USB Debugging)

* Using the data cable included with your Cellhasher, plug the Cellhasher into a computer and open Cellhasher Control.
  * A) If you ordered pre-installed, import your file
  * B) If you already have a file in .android you can copy that file and paste it Place the `adbkey.pub` and `adbkey` files in `C:\Users\[Your Username]\.android`
  * C) If you not, head to # each mobile device select "Always allow from this computer", you will only need to do this once per computer.
