# Set-Up (Download)

***

{% hint style="info" %}
#### Minimum System Requirements

**Computer**

* Operating System: macOS, Windows 10 (64-bit), Linux
* GPU: For screen mirroring 20 or more phones, a moderate performance graphics card is recommended. Some integrated graphics (APUs) have proven sufficient for light workloads.

**Phone**

* Operating System: Android 9 or higher
{% endhint %}

## Installing Cellhasher Control

{% hint style="info" %}
You may receive a warning prompt when opening the application. This is due to Cellhasher Control being in beta release and will be resolved at full release.
{% endhint %}

1. Download from the official GitHub:

<a href="https://github.com/CellHasher/Beta-Cellhasher/releases" class="button primary">Download Latest Release</a><br>

2. Under Assets, download the installer file that looks like this for your corresponding system: `cellhasher_controller_[version]_[architecture].[file-type]`
3. Run the installer and follow on-screen instructions.
4. Cellhasher Control is in beta, so stay updated by checking the GitHub releases page regularly for new builds.
