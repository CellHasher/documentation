# Devices

## Toolbar Functions:

<div align="left"><figure><img src="../../.gitbook/assets/IMG_1419 copy.png" alt=""><figcaption></figcaption></figure></div>

* Grid View: Switch to grid layout view of all devices
* Refresh: Update the list of connected devices
* Sleep Screens: Turn off displays of all connected devices while keeping them awake
* Settings: Access application configuration

***

## Device Functions:

<div align="left"><figure><img src="../../.gitbook/assets/IMG_1419 copy3.png" alt=""><figcaption></figcaption></figure></div>

* Device Info: View detailed specifications including:
  * CPU information
  * RAM capacity
  * Android version
  * Model details
  * Serial number
  * Custom device label (if set in Grid Screen)
* View Packages: Access installed applications on the device
* Mirror Screen: Launch a scrcpy window to view and control device screen (When troubleshooting, inspect [No Audio](../tabs-walkthrough/network.md#no-audio) settings)
* More Options (Context Menu):
  * Open ADB Terminal: Launch terminal for ADB commands
  * Open Fastboot Terminal: Launch terminal for Fastboot commands
